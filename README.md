# Vanilla Webinar Signup

A simple, clean webinar registration page built with **vanilla HTML** and **Bootstrap v1.3.0**.  

The page has a minimal **white-and-blue** theme and can be easily customized for any webinar or online event.

---

## Features

- Simple layout
- Easy to modify text and styles
- Vanilla HTML + Bootstrap v1.3.0 (no other dependencies)
- Minimal design for quick deployment
